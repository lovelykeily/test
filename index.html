인스타그램 데이터 파일을 업로드하세요

📦 데이터 파일 받는 법
  1. 인스타그램 앱 → 설정 및 활동 메뉴 진입
  2. 계정 센터 → 내 정보 및 권한 → 내 정보 내보내기
  3. 내보내기 만들기 → 기기로 내보내기 선택
  4. 정보 맞춤 설정 → 연결 활동: 팔로워 및 팔로잉 저장
  5. 형식: JSON 으로 설정 후 요청 → 다운로드
  6. 다운로드 받은 zip 파일 그대로 업로드

⚠️ 언팔하기 전에 꼭 해당 계정의 팔로우에서 내 계정이 없는지 확인 후 언팔하세요
